02 / DATA PLANE

Ingestion — where trust is established

Sovereignty is earned at ingest, not policed at query time. Every chunk leaves this pipeline carrying immutable metadata, and untagged data is unretrievable by construction — the retrieval filter requires the tag, so anything that skipped the classifier simply never surfaces.

04 / SOVEREIGNTY

Four sovereignties, not one

“UK sovereign” is really four separate guarantees. Name them separately or you will claim more than you deliver.

aData residency

Where the data physically sits.

All storage, vector DB, logs, backups and model weights in UK data centres. Default-deny egress; the “public cloud connectors” box is killed or geofenced — those are the paths that break sovereignty.

bData provenance

The “only surface UK data” guarantee — lives in the RAG layer.

Tag at ingest → constrain retrieval to a mandatory metadata filter → pin the model in-region → enforce via policy engine at the gateway. Route to a public OpenAI/Anthropic API and the guarantee is gone.

cOperational

Who can touch it.

UK-based, security-cleared operators; no foreign admin or root access to the control plane. Often overlooked, and a first-line question from government buyers.

dJurisdictional

Whose law can reach it.

UK legal entity; structured so no foreign statute (e.g. US CLOUD Act via vendor involvement) can compel disclosure. Backed by UK-held / customer-held keys (BYOK + HSM) and, ideally, GPU confidential compute — so even the operator can't read data under legal pressure.

Evidence buyers will ask for: ISO 27001 · SOC 2 · Cyber Essentials Plus · NCSC Cloud Security Principles · UK GDPR/DPA 2018 — plus DSPT (health) and a G-Cloud listing (public sector).

06 / DECIDE NEXT

Two calls that shape the build

Isolation model

Single- vs. multi-tenant. High-assurance buyers (gov, health) may need dedicated GPU nodes + a dedicated vector DB per tenant — which rewrites your unit economics.

Which model, in-region

An open-weights model you fully control is the cleanest sovereignty story. A “sovereign deployment” of a frontier model is stronger capability but a harder jurisdictional argument.
